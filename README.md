**CodeAOIS** is a multi-agent terminal OS designed for developers. It replaces scattered web interfaces with deeply integrated, context-aware command-line intelligence. 

[**View Live Demo**](https://codeaois.vercel.app/) ---

## 🚀 Features

- **Interactive Terminal Streams:** Real-time AI output streamed directly into your simulated command line.
- **Bento Grid Architecture:** A modern, asymmetrical feature showcase with physics-based hover effects.
- **Responsive Mobile Navigation:** Custom hamburger menu engineered for flawless touchscreen interaction.
- **Premium Micro-interactions:** Glowing custom mouse cursor, custom dark-mode scrollbars, and fluid accordion FAQs.
- **Dynamic OpenGraph Images:** Automated SEO banner generation for high-conversion social sharing.
- **CI/CD Pipeline:** Fully automated deployments via GitHub and Vercel.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

## 💻 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/codeaois.git](https://github.com/YOUR_USERNAME/codeaois.git)
   ```

2. **Navigate to the directory:**
   ```Bash
    cd codeaois
    ```
3. **Install the dependencies:**

   ```Bash
   npm install
   ```

4. **Start the development server:**

   ```Bash
   npm run dev
   ```
